# 📝 Descripción

Aplicación de escritorio desarrollada en **Java Swing** que implementa una calculadora con una interfaz moderna basada en **componentes gráficos personalizados**.

El proyecto demuestra buenas prácticas de desarrollo como:

* Implementación del patrón **MVC (Model–View–Controller)**
* Creación de **componentes visuales personalizados** utilizando `Graphics2D`
* Separación clara entre lógica de negocio, interfaz y controladores
* Código documentado con **Javadoc**

La aplicación sirve como ejemplo práctico de desarrollo de interfaces de escritorio modernas utilizando únicamente herramientas nativas de Java.

Puedes abrirlo en:
